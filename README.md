 # Material Downloader
 Multiplatform downloader made with Flutter(Dart)

# Platform Tested Compatibility

- [x] android
- [ ] web
- [ ] linux desktop
- [ ] windows desktop
- [ ] iOS
- [ ] fuchsiaOS
